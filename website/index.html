Features

Everything your phone can do,
now with AI

29 native Android tools, 21 pre-built skills, 10 cloud AI providers, and on-device Gemma 4 — all on your phone.

🛠️

29 Native Tools

SMS, calls, calendar, camera, contacts, file system, web browser, UI automation, screen capture, notifications, and more.

⚡

21 Pre-built Skills

Morning routine, email, messaging, navigation, social media, smart home, finance, health — ready to use out of the box.

🤖

10 Cloud + On-Device AI

Claude, GPT, Gemini, and 7 more cloud providers. Plus Gemma 4 runs locally — no API key, no internet.

🎙️

Voice Control

Talk to your phone naturally. Speech-to-text input and text-to-speech output for hands-free operation.

⏰

Scheduled Automation

Set up cron-style tasks that run automatically. Morning briefings, regular check-ins, recurring workflows.

🔒

Privacy First

All data stays on your device. Encrypted API key storage. No analytics, no tracking, no telemetry. You own everything.

Get Started

Up and running in 3 steps

Install

Download the APK from GitHub Releases and install it on your Android device. No account required, no server setup.

Configure

Add a cloud API key (Claude, GPT, Gemini) or download Gemma 4 to run AI entirely on-device — no key needed.

Command

Tell your phone what to do in plain language. MobileClaw handles the rest — tapping, typing, navigating.
