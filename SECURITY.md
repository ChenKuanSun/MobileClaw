# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MobileClaw, please report it responsibly.

**Email:** security@affiora.ai

**Do NOT** open a public GitHub issue for security vulnerabilities.

## Response Timeline

- **48 hours** — We will acknowledge receipt of your report
- **7 days** — We will provide a fix or mitigation plan for critical vulnerabilities
- **30 days** — We will publish a security advisory if applicable

## Scope

The following are in scope:
- Data exfiltration through tool abuse
- Prompt injection via skills or user input
- AccessibilityService misuse
- API key/token exposure
- SSRF or network-level attacks
- Permission bypass

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |
